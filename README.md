# Daybreak

An alarm app for iOS for UC Berkeley's Cubstart for Fall 2020

## Features

- Refreshed design
- Independent alarm volume separate from device volume
- Alarm wakeup games

### Planned

- Leaderboard
- Daily Digest
- Weather
